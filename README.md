# hand-gesture-recognition
Real-time hand gesture recognition using Python, OpenCV and MediaPipe
A real-time hand gesture recognition system built with Python, OpenCV and MediaPipe that detects hand landmarks through webcam and maps finger positions to custom messages and voice output.

#How it started
My exams had just ended and a friend suggested — let's build something. So the very next day I gave this a shot. No prior experience with computer vision, just basic Python knowledge and curiosity to figure things out.

#What it does

-Detects 21 hand landmarks in real-time through webcam
-Identifies which fingers are up or down
-Maps each gesture to a custom message on screen
-Speaks the gesture out loud using voice output

#Tech Stack
Python 3.14
OpenCV
MediaPipe

#What was difficult
Getting the setup right was the hardest part. Python version compatibility, finding the right MediaPipe version, understanding why certain libraries were not working — that alone took a significant chunk of the 6 hours I spent on this.

#What I learned
How computer vision works at a basic level
How MediaPipe tracks hand landmarks using coordinates
That understanding a library properly matters more than just installing it
Debugging is most of the work — and that is completely normal
Small projects like these build more clarity than hours of theory
